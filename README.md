# Bridge

## Sobre o Projeto

O **Bridge** é um projeto desenvolvido em equipe com foco em instituições de ensino, especialmente o SENAI.

A ideia surgiu a partir da dificuldade de organizar conteúdos, atividades e comunicação no dia a dia acadêmico. Pensando nisso, estamos criando uma plataforma que centraliza essas informações e facilita a interação entre alunos e professores.
O objetivo do projeto é tornar o ambiente de aprendizagem mais organizado, acessível e eficiente, usando a tecnologia como apoio direto no ensino.

---

## Objetivo

Criar uma aplicação web que funcione como uma ponte entre alunos e professores, reunindo em um só lugar:

* Conteúdos de aula
* Atividades e prazos
* Comunicação acadêmica
* Acompanhamento de desempenho

---

## Funcionalidades (em desenvolvimento)

* Organização de conteúdos por disciplina
* Cadastro e gerenciamento de atividades
* Comunicação entre alunos e professores
* Visualização de progresso do aluno
* Interface simples e fácil de usar
